## 🎵 About ArsMusic

**ArsMusic** is a premium, high-performance music streaming application designed for mobile devices. It provides a pure, seamless listening experience with a focus on a glassmorphic "Electric Night" aesthetic, ultra-smooth animations, and a robust offline-first architecture. Built for true music lovers.

---

## ✨ Key Features

- **🌐 Unlimited Streaming:** Access a massive library of global and local tracks dynamically sourced.
- **💾 Offline Downloads:** Native offline playback. Download songs to your local storage and listen anywhere without an internet connection.
- **🧠 Adaptive "For You":** The recommendation engine intelligently learns from your listening history. The more you listen, the more tailored your Home feed becomes!
- **📱 Native Mobile Feel:** Pull-to-refresh on the home feed, bottom sheet navigation menus, and swipeable elements that feel perfectly at home on an iOS or Android device.
- **🎨 Glassmorphic UI:** Deep dark mode with vibrant cyan accents and translucent, blurred surfaces.
- **🌙 Background Playback:** Fully supports native OS background audio and lock-screen media controls.
- **📂 Library Management:** Dedicated tabs for *Liked*, *Recent*, *History*, and *Offline* downloads.
